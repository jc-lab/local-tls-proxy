# local-tls-proxy

```text
Usage of local-tls-proxy
  -ca-cert string
        KeyPair certificate file path (default "ca.pem")
  -ca-key string
        KeyPair key file path (default "ca.key")
  -port int
        server port (default 5443)
```

Proxing https://{PORT}.{IP}.sslip.io:5443/ to http://{IP}:{PORT}/
